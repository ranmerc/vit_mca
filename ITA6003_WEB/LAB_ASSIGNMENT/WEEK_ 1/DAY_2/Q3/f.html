Web Programming

HTML
HTML stands for HyperText Markup Language. It is used to create web pages.
CSS
CSS stands for Cascading Style Sheets. It is used to style web pages.
JavaScript
JavaScript is a programming language that is used to make web pages interactive.
PHP
PHP stands for Hypertext Preprocessor. It is a server-side scripting language that is used to create dynamic web pages.
MySQL
MySQL is a relational database management system that is used to store data.
